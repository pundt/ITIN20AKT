LadeAlleReisen

Lädt alle Reisen, die den Filterkriterien entsprechen
LadeReiseZeitpunkte
Lädt zu einer Reise alle Zeitpunkte dieser Reise (also alle Reisedaten)
LadeUnterkunftZuReise
Sucht anhand einer Reise_ID die dazugehörige Unterkunft
ReiseEntfernen
Entfernt eine Reise und alle zugehörigen Daten und gibt true zurück, falls alles geklappt hat
Restplätze
Ermittelt die Restplätze einer Reise zu einem speziellen Datum aus der DB
SpeicherReise
Speichert eine Reise in der Datenbank
SpeicherReiseAnzahl
hier wird die ReiseAnzahl festgelegt / wird so oft wie freie Plätze aufgerufen und nur das Reisedatum eingetragen
SpeicherReiseDatum
hier wird ein Datum gespeichert und die neue Id davon zurückgeliefert
SucheReise(Int32)
Sucht eine Reise anhand einer ID aus der Datenbank
SucheReise(String)
Sucht nach einer spezifischen Reise mittels eines Suchtexts
SucheReisedatum
Sucht ein Reisedatum anhand der Reisedatum_ID und liefert dieses zurück
SucheReiseZuDatum
Sucht eine Reise anhand einer Reisedatum_ID und liefert diese zurück
